# 📰 Fake News Detection System

A Machine Learning-based application that classifies news articles as **Fake ❌** or **Real ✅** using Natural Language Processing (NLP).

## 🚀 Features

* Detects fake vs real news
* NLP-based text preprocessing
* Fast and simple predictions
* Easy to run and deploy

## 🛠️ Tech Stack

* **Python**
* **Scikit-learn, Pandas, NumPy, NLTK**
* **Flask (optional frontend/backend)**

## 📂 Project Structure

bash
fake-news-detection/
│── data/
│── model/
│── templates/
│── static/
│── app.py
│── requirements.txt

## How It Works

1. Preprocess news text
2. Convert to features (TF-IDF)
3. Train ML model
4. Predict Fake or Real


## Installation

bash
git clone https://github.com/your-username/fake-news-detection.git
cd fake-news-detection
pip install -r requirements.txt
python app.py


## ▶️ Usage

Enter news text → Click predict → Get result (**Fake/Real**)
